Fale Conosco

Eu pedi pra fazer três parágrafos na atividade 1.1, um título e uma imagem. O index continua precisando de três parágrafos, título a gente agora vai ter um monte e a imagem vocês vão manter a do index.html mas trazendo agora a imagem baixada.

Agora especificamente sobre os títulos, vocês precisam fazer um menu onde pela hierarquia de títulos deve ser utilizado do h1 até o h3. Todos os itens do menu serão clicáveis e devem redirecionar a uma página com o mesmo nome do item do menu. Lembrando, sempre, de fazerem itens de acordo com os seus respectivos temas.

A gente não mexeu com nada de CSS ainda, então vai ficar tudo bem feio. O menu inclusive fica horroroso, porque não indenta os subtópicos e tals, mas a ideia é ficar feio mesmo pra gente ver a coisa evoluindo nas entregas posteriores.

Ah, tem uma coisa que pode ajudar vocês a pontuar melhor nessa atividade. Existe uma função no github chamada pull, onde você pode baixar um repositório do GitHub, então se vocês fizerem isso no meu repositório vocês poderão rodar o meu site na máquina de vocês. Isso vai ajudar vocês a verem como deve ficar de fato uma entrega correta da atividade 1.2. Mas sobre como utilizar o pull fica por conta de vocês estudarem.
